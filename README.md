## Software Engineer

I’m a backend-focused developer who enjoys solving problems, learning quickly, and building software that turns ideas into real products. With a background that began in statistics and data science, I found myself drawn more to coding, system design, and implementation than analysis alone, which led me into software development and a strong interest in backend logic, application flow, and end-to-end product building. I enjoy turning complex requirements into clear, maintainable solutions, collaborating with teams, adapting quickly to new tools and environments, and using AI-assisted development to prototype faster and solve technical challenges more effectively.

- 🌍 I'm based in Vancouver, BC
- 🖥️ See my portfolio at [zipeng.vercel.app](http://denis-liu.vercel.app/)
- ✉️ You can contact me at [zipeng.liu.mail@gmail.com](mailto:denis.liu.mail@gmail.com)
- 👥 I'm looking to collaborate on exciting ideas, impactful projects, and new challenges
- ⭐ When I’m not building, you’ll probably find me playing table tennis, travelling, or cooking.
